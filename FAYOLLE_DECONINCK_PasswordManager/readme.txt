L'utilisateur peut s'inscrire en cliquant sur "Pas encore inscrit ?".
Une fois inscrit, il doit rentrer ses identifiants (sur notre site) puis choisir entre deux options : 
- ajouter des couples identifiants/mots de passe pour un site particulier
- afficher ce couple en rentrant l'url de ce site

Dans notre projet, le mot de passe des diférents sites est chiffré au niveau du frontend et est enregistrée dans la dans la base
de données no SQL.
L'identification par challenge est fonctionnelle.
Seul le déchiffrage du mot de passe n'est pas fonctionnel.

Pour lancer le serveur, exécutez la commande "node index.js"
Pour vous , exécutez la commande "node
